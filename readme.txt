Moolah provides you a single-page checkout on any website.

== Description ==

Moolah E-Commerce is a online store service offering a simplified checkout for simple, premium goods. The user selects the item they want, provides contact/shipping information, and complete the purchase. Done. Everything happens within a single page.

The Moolah E-Commerce Plugin automates the loading of your Moolah store. Simply enter your Store ID with optional category and product IDs, and the plugin will load your store into your WordPress blog.


== Installation ==

Upload the Akismet plugin to your blog, Activate it, then enter your [Moolah Store ID](http://moolah-ecommerce.com/sign-up).
