Moolah provides you a single-page checkout on any website.

== Description ==

Moolah E-Commerce is a online store service offering a simplified checkout for simple, premium goods. The user selects the item they want, provides contact/shipping information, and completes their purchase. Done. Everything happens within a single page - there is no need to bounce the customer to another website.

The Moolah E-Commerce Plugin automates the loading of your Moolah store. Simply enter your Store ID with optional category and product IDs, and the plugin will load your store into your WordPress website.


== Installation ==

Upload the Moolah plugin to your blog, Activate it, then enter your [Moolah Store ID](http://moolah-ecommerce.com/sign-up).  For testing, you can use the ID 2642953450.

== Screenshots ==

1. Moolah is designed to streamline the sale of simple products, and supports basic features such as Product Variation with unique prices.
2. You can let the user customize a good or service using a variety of form attributes which are added to the order. Attributes with costs change the order total.
3. The store is controlled with the Management Council.  Changes that you make here are automatically propagated to all instances of your store, whether on WordPress or another website.
4. Moolah supports all the basic features that you would expect from a professional e-commerce system, such as Taxes and Fees, Coupons and Promotions, Shipping and Delivery, and etc.....
